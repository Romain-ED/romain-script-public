Vonage Numbers Manager v2.1.0 - Multi-User Web Interface

This web application simplifies the management of Vonage phone numbers through an intuitive, multi-user interface.

Key Features:
  • Multi-User Support: Each user provides their own credentials (no shared storage)
  • Account Dashboard: View balance and subaccounts at a glance
  • View Owned Numbers: Complete listing of your phone numbers
  • Search Available Numbers: Find new numbers with advanced filtering
  • Purchase Numbers: Buy numbers with subaccount assignment
  • Cancel Numbers: Remove unwanted numbers (irreversible)
  • Real-time Logging: Live activity monitoring via WebSocket
  • Responsive Design: Works perfectly on desktop and mobile devices

Getting Started

1. Connect Your Account
  1. Enter your Vonage API Key and Secret
  2. Check "Save credentials" to store them locally
  3. Click "Connect Account"
  4. Your phone numbers will load automatically
2. View Your Numbers

After connecting, all your owned numbers appear in the "Your Phone Numbers" section with details like country, type, features, and application ID.

3. Search for New Numbers
  1. Enter a country code (e.g., US, GB, FR)
  2. Select number type and features
  3. Set results limit (1-100)
  4. Click "Search Available Numbers"
4. Purchase Numbers
  1. Select desired numbers using checkboxes
  2. Click "Buy Selected Numbers"
  3. Review costs and optionally assign to subaccount
  4. Confirm purchase
5. Cancel Numbers
  1. Select numbers to cancel in your owned numbers
  2. Click "Cancel Selected"
  3. Read warning carefully - this action is irreversible
  4. Confirm cancellation

Detailed Features

Multi-User Session Management
  • Session-based Credentials: Each user provides their own API credentials (no shared storage)
  • Secure Isolation: Multiple users can work simultaneously without credential conflicts
  • Auto-disconnect: Sessions expire automatically for security
  • No Persistent Storage: Credentials are cleared when session ends
Account Dashboard
  • Real-time Balance: Live account balance display with currency formatting
  • Credit Limits: Visual indicators for account credit status
  • Subaccount Overview: Complete list of all subaccounts with details
  • One-click Refresh: Update account information instantly
Comprehensive Number Management
  • Complete Inventory: View all owned numbers with full details (country, type, features, applications)
  • Bulk Selection: Select all/individual numbers for batch operations
  • Live Counter: Real-time count of total owned numbers
  • Instant Refresh: Update number list with latest information from Vonage
  • Application Tracking: See which applications are assigned to each number
Advanced Number Search
  • Global Coverage: Search numbers from any country using ISO 3166-1 codes
  • Type Filtering: Filter by landline, mobile, or toll-free numbers
  • Feature Combinations: Search for specific feature sets (SMS, VOICE, MMS combinations)
  • Flexible Results: Configurable result limits (1-100 numbers)
  • Price Transparency: Display both initial purchase cost and monthly fees
  • Smart Selection: Select all or individual numbers for purchase
Intelligent Purchase System
  • Cost Calculator: Automatic calculation of total initial and monthly costs
  • Subaccount Assignment: Assign numbers to specific subaccounts during purchase
  • Purchase Preview: Review all selected numbers before confirming
  • Bulk Purchase: Buy multiple numbers in a single transaction
  • Transaction Results: Detailed success/failure reports for each number
  • Balance Validation: Check account balance before purchase
Safe Number Cancellation
  • Multi-step Confirmation: Multiple warning dialogs to prevent accidental cancellations
  • Impact Warning: Clear explanation of cancellation consequences
  • Bulk Cancellation: Cancel multiple numbers simultaneously
  • Detailed Results: Complete report of successful/failed cancellations
  • Immediate Effect: Numbers are cancelled instantly and removed from inventory
Real-time Activity Monitoring
  • Live WebSocket Logging: Real-time activity updates without page refresh
  • Complete Audit Trail: All API calls, responses, and errors logged with timestamps
  • Color-coded Messages: Visual distinction between info, success, error, and warning messages
  • Auto-scroll Control: Toggle automatic scrolling to follow latest activity
  • Log Management: Clear logs on demand to reduce clutter
  • Error Details: Comprehensive error information for troubleshooting
Cross-platform Compatibility
  • Responsive Design: Optimized for desktop, tablet, and mobile devices
  • Universal Browser Support: Works with Chrome, Firefox, Safari, Edge
  • Touch-friendly Interface: Large buttons and touch targets for mobile use
  • Flexible Layout: Adapts to different screen sizes and orientations
  • Keyboard Navigation: Full keyboard accessibility support
Security & Reliability
  • No Cloud Storage: All operations performed locally with direct API calls
  • Input Validation: Comprehensive validation of all user inputs
  • Error Recovery: Graceful handling of network failures and API errors
  • Rate Limiting Awareness: Respects Vonage API rate limits
  • Connection Monitoring: Automatic detection of connectivity issues

Troubleshooting

Connection Issues
  • Invalid credentials: Verify API key and secret
  • Network errors: Check internet connection
  • Account issues: Ensure account is active
Search Problems
  • No results: Try different country codes or filters
  • Invalid country: Use 2-letter ISO codes (US, GB, etc.)
Purchase/Cancel Failures
  • Insufficient balance: Check account balance
  • Permission denied: Verify account permissions
  • Number unavailable: Number may have been taken
Getting Help

Check the Activity Log for detailed error messages. For Vonage API issues, consult the official documentation or contact Vonage support.
