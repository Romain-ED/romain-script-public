# 🚀 Vonage Numbers Manager - Quick Start

Transform your Vonage phone number management with this modern web interface!

## 📦 What You Get

✅ **Modern Web Interface** - Beautiful, responsive design that works everywhere  
✅ **Real-time Logging** - Live activity updates via WebSocket  
✅ **Bulk Operations** - Purchase/cancel multiple numbers at once  
✅ **Secure Credentials** - Local storage with proper warnings  
✅ **Cross-platform** - Works on Windows, macOS, Linux, tablets, phones  
✅ **No Installation** - Just a web browser needed  

## ⚡ 60-Second Setup

### Option 1: Super Quick (Automated)
```bash
# 1. Download all files to a folder
# 2. Run this one command:
python setup.py

# 3. Start the app:
python main.py
# OR double-click: run_web_interface.bat (Windows)
# OR run: ./run_web_interface.sh (Mac/Linux)

# 4. Open browser: http://localhost:8000
```

### Option 2: Manual (If you prefer control)
```bash
# 1. Install dependencies
pip install fastapi uvicorn jinja2 requests pydantic python-multipart

# 2. Create folders
mkdir static templates logs

# 3. Place files:
#    main.py (root folder)
#    templates/index.html
#    static/styles.css
#    static/app.js

# 4. Start server
python main.py

# 5. Open: http://localhost:8000
```

### Option 3: Docker (For production)
```bash
# 1. Build and start
docker-compose up --build

# 2. Open: http://localhost:8000
```

## 🎯 First Steps

1. **Enter Credentials**: Your Vonage API Key & Secret
2. **Click Connect**: Your numbers load automatically
3. **Search Numbers**: Enter country code (US, GB, etc.)
4. **Purchase**: Select numbers → Buy → Confirm
5. **Cancel**: Select owned numbers → Cancel → Double confirm ⚠️

## 🚨 Quick Safety Notes

- **Purchases cost real money** - review carefully!
- **Cancellations are IRREVERSIBLE** - be absolutely sure!
- **Credentials are stored locally** - basic encoding, not encryption
- **Test everything first** in a safe environment

## 🆘 Quick Troubleshooting

| Problem | Quick Fix |
|---------|-----------|
| "Connection refused" | Run `python main.py` first |
| "Port 8000 in use" | Change port in main.py |
| "No module named..." | Run `pip install -r requirements.txt` |
| "Invalid credentials" | Double-check API key/secret |
| "No numbers found" | Try different country codes (US, GB, FR) |
